# Searchable Select

# How to use

```html
<select id="select_id">
   
</select>
```

You need script like this:


```javascript
<script>
 
  
  $(“#select_id”).searchableSelect({url:"https://github.com/jiuli/searchableSelectAjax/demo.json"});即可调用远程数据
</script>
```
