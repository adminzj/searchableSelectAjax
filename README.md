# Searchable Select

# How to use

```html
<select id="select_id">
   
</select>
```

You need script like this:


```javascript
<script>
 
  
  $(“#select_id”).searchableSelect({url:"https://github.com/jiuli/searchableSelectAjax/demo.json"});
  #加上url参数，即可调用远程数据，后台代码对应写上search_name搜索字段
  
</script>
```
